jsPlumbToolkit

Welcome to the Toolkit edition of jsPlumb. To get started, you'll need these things installed:

  • NodeJS (version 5 or higher)
  • npm (this is installed with NodeJS)
  • Grunt (after installing Node, `npm install grunt-cli -g` will install this)
Once you have these all installed, you'll need to run this command: 
    npm install

This will iterate through every demonstration and run `npm install`, `grunt webpack` for those demos that use Webpack, and `npm run tsc` for Angular 2 demos using Typescript.

Once you've installed everything, you can either host this entire directory on a webserver somewhere, or you can use the embedded webserver:

Mac/Linux/Unix

   npm start

Windows

    npm start_windows
By default, this will start up a webserver on port 8910.

Getting Started

With Grunt you can get started straight away with a copy of an existing demo, or with an almost empty app you can use as a template.

Cloning an existing demo

grunt clone --app=database-visualizer --o=<your output directory>
Valid values for `-app` are:
  • database-visualiser
  • flowchart-builder
  • absolute
  • circular
  • hierarchy
  • multiple-hierarchy
  • paths
  • hello-world
  • multiple
  • skeleton
  • spring
  • groups
  • connectivity
  • angular-1.x
  • angular
  • angular-cli
  • angular2
  • angular2-cli
  • angular-1.x-groups
  • angular-groups
  • react
  • vue
  • webpack

Creating a new skeleton app

grunt create --o=<your output directory>
