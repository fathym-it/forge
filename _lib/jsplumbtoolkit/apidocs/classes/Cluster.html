    Cluster Class

    A cluster of connected Nodes/Groups. Currently a basic implementation; we can in future expand the API of this object to provide more information about the cluster such as edge count, interdependency etc.

    Clusters are not created directly. The internal API creates these in response to a getClusters call on a Graph.
