Welcome to the Fathym Forge

As technology continues to shift and grow, a gap has emerged in how Enterprises can leverage a community of experienced and skilled developers. More important, is how to leverage this community in a scalable and secure way that enables both sides to thrive in the delivery of business-critical solutions and products.

The Fathym Forge supports a configuration driven development experience that separates developers from the technology infrastructure configured by enterprise engineers. This enables developers to create solutions against an application spec that enterprise managers can configure to their platform.

Let's Get Started

The Forge is a collection of Low Code development workflows for building rich Data Applications in a distributed, secure, and scalable architecture. As we continue to roll out our Low Code tool sets, we will also be adding additional infrastructure providers. For now, we have partnered with Microsoft Azure to deliver a best-in-class architecture for applications built with the Forge.

Before digging in to the features of the Forge, you'll need to make your first infrastructure decision. Are you just Exploring the features, looking for support with a managed Azure Subscription, or do you have your own Azure accounts setup?

The best and easiest way to get started is with your own Azure Subscriptions. There are great cost saving opportunities for new users and you only ever pay for what you need. To get started with a new Azure Account, head over here!
